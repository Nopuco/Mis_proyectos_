# 🛡️ Auditor Automatizado de Cumplimiento (Compliance-as-Code)

Este repositorio es un **Bot de Auditoría Proactiva** diseñado para detectar riesgos de **Ciberseguridad y Fraude** en infraestructura antes de su despliegue.

## 🚀 Innovación
Utiliza **Checkov** para validar políticas basadas en estándares internacionales:
*   **ISO 27001 / ISO 42001** (Gobernanza de Datos e IA).
*   **NIST & CIS Benchmarks**.
*   **PCI-DSS** (Prevención de fraude financiero).

## 🛠️ Cómo funciona
El bot analiza archivos de Terraform (`.tf`) buscando configuraciones inseguras, como buckets de datos públicos o falta de cifrado.
